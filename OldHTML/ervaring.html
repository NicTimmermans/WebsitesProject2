Mijn eerste werk ervaring in de IT-sector was Soulmate in Antwerpen. Dit bedrijf maakt VR en AR-applicaties op maat. Ik was hier begonnen als stagiair nadat ik een opleiding .Net C# bij de VDAB had gevolgd.
Ik heb hier leren werken als front-end developer in Unity, een game engin waarmee ik verschillende XR-applicaties gemaakt heb. Na een half jaar stage en een half jaar IBO ben ik aangenomen als Unity developer en heb ik nog een jaar bij Soulmade gewerkt.
Dit was een klein team met een leuke sfeer. Ik heb veel bijgeleerd van mijn collega's over de professionele IT-wereld.
Na deze werk ervaring heb ik besloten om mijn graduaat programmeren te halen om mijn profiel of software developer nog aantrekkelijker te maken.

Een van de eerste applicaties die ik heb ontwikkeld was BirthVR. Dit is een VR-meditatie experience voor zwangere vrouwen. Deze wordt ook gebruikt in bepaalde ziekenhuizen in Nederland om vrouwen voor te bereiden en te begeleiden tijden hun bevalling.
Ik heb ook een AR-applicaties gemaakt dat een digitale warmtepomp in de ruimte kan plaatsten en de werking hiervan in animaties kan uitleggen. Hiervoor heb ik ook een API en CMD gemaakt.
Een andere applicatie waar ik een prototype voor gemaakt heb ik een AR-tool voor zwembadbouwers dat een digitaal zwembad in de tuin van hun klanten kan zetten.
De meest uitdagende applicatie die ik ontwikkeld heb was een las-simulatie in VR. Hiervoor werkte ik met fluid simulations, voxel grids en mesh generators.
